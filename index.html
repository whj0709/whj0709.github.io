Clean Table Exporter

A lightweight Chrome extension to export clean, structured tables from web pages.

Local-first: Runs entirely in your browser. No account required.

Fast workflow: Activate → Select a table → Export as CSV / Excel.

How it works

  1. Open a web page containing a table (or table-like layout).
  2. Open the extension and click to select the table.
  3. Export to CSV or Excel.

Free & Pro

Free users can export once per day. Upgrade to Pro to unlock unlimited exports with a one-time purchase.
